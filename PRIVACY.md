# 隐私政策

## USpeedo Email 插件隐私政策

### 数据收集和使用

USpeedo Email 插件（以下简称"本插件"）是一个用于在 Dify 工作流中发送邮件的工具插件。本插件的数据处理方式如下：

#### 1. 凭证信息

- **收集的数据**：本插件需要您提供 USpeedo 的访问凭证（ACCESSKEY_ID 和 ACCESSKEY_SECRET）
- **存储方式**：凭证信息仅存储在您的 Dify 实例中，采用加密方式保存
- **使用目的**：用于向 USpeedo API 进行身份验证，以发送邮件
- **数据共享**：凭证信息不会与第三方共享，仅用于与 USpeedo 官方 API 通信

#### 2. 邮件内容

- **收集的数据**：邮件发送过程中，本插件会处理以下信息：
  - 发件人邮箱地址
  - 收件人邮箱地址
  - 邮件主题
  - 邮件内容（HTML 格式）
  - 发件人显示名称（如提供）
- **数据传输**：上述信息会通过 HTTPS 加密传输到 USpeedo API 服务器
- **数据存储**：邮件内容由 USpeedo 服务提供商处理，本插件不存储任何邮件内容
- **使用目的**：仅用于发送您指定的邮件

#### 3. 第三方服务

本插件使用 USpeedo 邮件服务来发送邮件。当您使用本插件时，您的邮件数据会传输到 USpeedo 的服务器。我们建议您查看 [USpeedo 的隐私政策](https://uspeedo.com) 以了解他们如何处理您的数据。

### 数据安全

- 所有 API 通信均通过 HTTPS 加密传输
- 凭证信息在 Dify 平台中采用加密存储
- 本插件不会记录或存储任何邮件内容或收件人信息

### 数据保留

- 本插件本身不存储任何邮件数据
- 凭证信息存储在您的 Dify 实例中，您可以随时删除或更新
- 邮件发送记录由 USpeedo 服务管理，请参考 USpeedo 的数据保留政策

### 用户权利

您有权：
- 随时访问和更新您的凭证信息
- 删除插件配置，这将停止使用本插件
- 查看 Dify 平台中的插件使用日志
